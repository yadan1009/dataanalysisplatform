### ğŸ“¦ Installation & Configuration

#### Requirements
- Python 3.8+

#### 1. Clone the Project

#### 2. Create Virtual Environment
```bash
python -m venv .venv
```


#### 5. Start the Application
```bash
python app.py
```

Visit `http://localhost:8080` to use the platform.
